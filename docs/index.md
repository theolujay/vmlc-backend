# Welcome to the VMLC API Documentation

Welcome to the official documentation for the Verboheit Mathematics League Competition (VMLC) API. This hub provides all the information you need to successfully integrate with and utilize our platform.

The VMLC API is a robust backend service designed to manage every aspect of the competition, from user registration and verification to exam administration, real-time scoring, and leaderboard tracking.

## Key Sections

- **Getting Started**: If you're new here, this is the place to begin. It covers everything from obtaining authentication credentials to making your first API call.
- **API Reference**: A detailed breakdown of every available endpoint, including request parameters, response payloads, and required permissions.
- **User Roles**: An explanation of our role-based access control (RBAC) system, detailing the abilities of candidates and staff members.
- **Advanced Topics**: For experienced users, this section covers more complex subjects like error handling, rate limiting, and API versioning.
