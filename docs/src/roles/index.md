# User Roles & Permissions

The VMLC API uses a role-based access control (RBAC) system to ensure that users can only access the features and data appropriate for their position. This section provides a detailed overview of the different user roles and the permissions associated with each one.

Understanding these roles is essential for building secure and efficient integrations.
