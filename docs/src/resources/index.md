# Resources

This section contains additional resources and supplementary materials to help you make the most of the VMLC API. From tracking recent changes to getting help, these documents provide valuable context and support.

## What's Inside

- **[Changelog](changelog.md)**: A detailed log of all changes, updates, and new features introduced to the API. Keep track of the latest developments and version histories here.

- **[Interactive Documentation](interactive-documentation.md)**: Explore and interact with the API in real-time using our Swagger UI and ReDoc interfaces. This is a great way to test endpoints without writing any code.

- **[Support](support.md)**: Need help? This page provides information on how to get in touch with our support team for technical assistance or to ask questions.
