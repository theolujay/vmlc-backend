# API Reference

This section provides a complete and detailed reference for every endpoint available in the VMLC API. Each entry includes the endpoint's path, required permissions, request parameters, and example response payloads.

Whether you're managing users, administering exams, or retrieving leaderboard data, this is the place to find all the technical details you need.
